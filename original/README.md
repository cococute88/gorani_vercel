# cococute88-opt-gorani-finance-v2

Streamlit 기반 자산/배당 관리 앱입니다.

## 네비게이션 구조
- **상단 가로 메뉴**: 주요 화면으로 빠르게 이동하는 퀵 네비게이션
- **사이드바 메뉴**: 기존 `st.navigation` + 로그인 상태/자동저장/즐겨찾기 등 운영 UI 유지

즉, 상단과 사이드바를 병행해 사용합니다.

## 성능 최적화 1차 (저위험)
- 배당 캘린더 외부 API 호출 경로에 세션 재사용/캐시를 보강했습니다.
- 순수 계산(배당 주기 추론, 이벤트 투영)에 TTL 캐시를 적용했습니다.
- Firebase client 초기화를 `st.cache_resource`로 안정화했습니다.
- 사용자별 데이터 캐시, Firebase 스키마/경로, 자동저장 타이밍, 계산 공식은 변경하지 않았습니다.
- 자세한 내용: `docs/PERFORMANCE_AUDIT.md`
